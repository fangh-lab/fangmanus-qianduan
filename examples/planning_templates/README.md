# 规划模板示例

这里包含了一些规划模板示例，可以在 Web UI 中作为 "Planning Template" 上传使用。

## 网页探索 Demo

**文件**: `web_exploration_demo.txt`

这是一个专门用于测试浏览器功能的网页探索任务规划模板。

### 使用方法

1. 在 Web UI 的 "Planning Template" 字段中，选择上传 `web_exploration_demo.txt`
2. 在 "Prompt" 中输入一个网页探索任务，例如：
   - "探索 GitHub 网站的主要功能和特点"
   - "访问并分析某个新闻网站的结构"
   - "探索某个在线工具网站的功能"

### 特点

- **快速执行**：规划步骤限制在 3-5 步内
- **智能工具选择**：优先使用轻量级 web_search，仅在需要交互时使用浏览器
- **结果导向**：每个步骤都有明确的输出目标
- **流畅体验**：减少不必要的中间过程，快速看到结果

### 测试建议

建议使用以下类型的任务来测试：

1. **简单信息查询**：如"搜索某个技术的最新发展"
2. **网站探索**：如"探索某个网站的功能和特点"
3. **内容提取**：如"从某个网页提取关键信息"

这些任务可以很好地测试：
- Web 搜索功能
- 浏览器自动化功能
- 步骤确认交互
- 结果展示
